# nebula-engine

[![Smoke tests](https://github.com/owendavies413-netizen/nebula-engine/actions/workflows/smoke.yml/badge.svg)](https://github.com/owendavies413-netizen/nebula-engine/actions/workflows/smoke.yml)

Small HTTP tunneling/proxy helper used for Ultraviolet-compatible proxies.

Usage
-----

- Start server:

```bash
npm start
```

- Run smoke tests locally:

```bash
npm test
```

Enable debug logs with `BARE_DEBUG=1`.
